# Templates for basic repository files

This folder contains templates for basic repository files, which you can grab when you want to create a new project. All you have to do is to make appropriate changes within them according to your project's needs, rename them to the same name but without the prefix `tpl-` and place them in the root folder of your project. For example, `/templates/tpl-CHANGELOG.md` becomes `/CHANGELOG.md`.

Each template file starts with headings and text detailing the layout that particular file should follow. At the end of the file, I've written a set of instructions you should go through that helps you to know which changes you should make.
